## Running Tests

**Run all tests**
```
npm test
```

**Run tests by name**
```
npm test -- -t 'LinkedList'
```

**Run tests from specific file**
```
npm test -- ./src/data-structures/linked-list/__test__/LinkedList.test.js
```
